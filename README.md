# Public Presentations

This repository contains presentations that I have delivered in public contexts. It will probably be **easier to navigate the [GitHub Pages](https://jreades.github.io/talks/) version** of this site rather than browse the repo directly. 

In the interests of promoting open science I have made these publicly accessible; however, please note that everything here is (unless otherwise indicated) under a CC BY-NC-SA 4.0 license, meaning that commerical use (in the sense of talks given 'for pay', I don't mind if you want to include one in an unpaid talk) is not allowed and attribution must always be given. 

In some small number of cases I may have removed imagery or other information at the request of a project partner, though I have tried to avoid this whenever possible.

I found [this page](https://danielabaron.me/blog/build-and-publish-presentation-with-html-and-css/) useful for thinking through some of the technical issues with publishing Reveal.js presentations from Quarto over to GitHub.
